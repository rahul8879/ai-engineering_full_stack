## Section 1: Python Basics

### Problem 1: Hello, AI World!
Write a program that asks the user for their name and prints:  
`Hello, <name>! Welcome to your AI Engineering journey.`

### Problem 2: Data Type Detective
Create variables of the following types: `int`, `float`, `str`, and `bool`. Print each variable along with its type using `type()`.  
**Example output:** `42 is of type <class 'int'>`

### Problem 3: Simple Calculator
Ask the user for two numbers. Print their:
* Sum
* Difference
* Product
* Quotient 
> **Note:** Handle division by zero with a friendly error message.

## Section 2: Conditionals (if / elif / else)

### Problem 4: AI Model Grader
Ask the user to enter a model accuracy score (0–100). Print a rating based on the score:
* **90–100** → "Excellent! Production-ready."
* **75–89** → "Good. Needs minor tuning."
* **60–74** → "Fair. More training data needed."
* **Below 60** → "Poor. Re-evaluate your approach."

### Problem 5: Even or Odd Classifier
Ask the user for a number and:
1. Print whether it is **even** or **odd**.
2. Print whether it is **positive**, **negative**, or **zero**.

### Problem 6: Login System (Mini)
Store a username and password string in your code. Ask the user to enter both. Print `"Access granted!"` if correct, or `"Access denied."` if either is wrong.

## Section 3: Loops

### Problem 7: Countdown Timer
Using a `while` loop, print a countdown from 10 to 1, then print:  
`"🚀 Model training started!"`

### Problem 8: Multiplication Table
Ask the user for a number and print its multiplication table from 1 to 10 using a `for` loop.  
**Example:** `7 x 3 = 21`

### Problem 9: Sum of a List
Given the list `[4, 17, 2, 9, 33, 7, 15]`, use a loop to:
1. Print each number.
2. Calculate and print the total sum.
3. Find and print the largest number (**without** using the built-in `max()` function).

### Problem 10: FizzBuzzAI 🤖
Loop through numbers 1 to 50. For each number:
* Print **"Fizz"** if divisible by 3.
* Print **"Buzz"** if divisible by 5.
* Print **"FizzBuzz"** if divisible by both.
* Otherwise, print the number.

* After each wrong guess, hint **"Too high!"** or **"Too low!"**.
